# Running Simplified V2 Application

- git clone the repository

```
$ git clone https://github.com/ChoiceCoin/v2.git
```

- go into the MVP directory

```
$ cd MVP
```
- install dependencies

```
$ npm install 
```

- start v2 applicatiion in development

```
$ npm run dev
```
